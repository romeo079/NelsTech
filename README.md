# NelsTech Solutions

NelsTech Solutions is a professional business website developed using PHP, MySQL, HTML, CSS, JavaScript, and PHPMailer. The system allows customers to request quotations, send messages, and contact the company for installation services.

## Services Offered

- Air Conditioning Sales & Installation
- Solar Panel Installation
- CCTV Installation
- Gate Motor Installation
- Electric Fence Installation

---

## Features

### Customer Features

- Modern responsive website
- Home page
- About Us page
- Services & Quote Request page
- Gallery page
- Contact page
- WhatsApp contact button
- Quote request submission
- Contact form submission
- Email notifications
- Mobile-friendly design

### Admin Features

- Secure admin login
- View quote requests
- View customer messages
- Delete quotes
- Delete messages
- Reply to customer emails
- Download quotation PDFs
- Manage gallery images (future enhancement)
